# 🧩 拼豆AI图纸生成器

将任意图片或文字描述转换为拼豆图纸，支持多种板型、色系和交互功能，帮你从创意到成品一步到位。

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Windows / macOS / Linux

### 安装与启动

```bash
# 1. 进入项目目录
cd perler-app

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动服务
python app.py
```

启动后浏览器访问 **http://127.0.0.1:5000** 即可使用。

### 生产环境部署（可选）
```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

## ✨ 功能一览

### 基础功能
| 功能 | 说明 |
|------|------|
| 📷 图片转图纸 | 上传 JPG/PNG/GIF，自动缩放匹配板型，智能映射拼豆色号 |
| 🤖 AI文字生成 | 输入描述（如"可爱的猫咪"），AI生成图片后自动转图纸 |
| 📐 多板型支持 | 52钉板（入门）、78钉板（经典）、104钉板（复杂图案） |
| 🎨 5种色系套装 | 72色 / 96色 / 120色 / 144色 / 221色，按需选择 |
| 📤 多格式导出 | PNG（带网格色号）、PDF（A4排版含用量清单）、JSON（二次开发） |
| 📊 用量统计 | 实时统计每种颜色使用颗数，按用量排序，显示百分比和条形图 |

### 进阶功能（V2）
| 功能 | 说明 |
|------|------|
| 🪞 镜像翻转 | 一键水平翻转图纸，适合对称图案制作 |
| 🎯 单色高亮 | 点击格子或色号，高亮同色格子，其余变暗，柔和发光效果 |
| ✅ 按色号标记进度 | 按颜色勾选完成状态，已完成色号格子显示斜线纹理，制作流程更自然 |
| 📋 材料清单面板 | 按色系分组显示所有色号用量，含勾选框，一目了然 |
| 🧹 去杂色 | 三级去杂：轻度（孤立点）、中度（≤2格连通区）、重度（≤3格连通区） |
| 🔍 大图缩放+小地图 | 滚轮/捏合缩放，拖拽平移，右下角小地图快速定位 |
| 🔄 一键品牌转换 | 生成后切换色系，基于原RGB重新匹配，图案形状不变 |

### 体验优化
| 优化 | 说明 |
|------|------|
| 📏 5格辅助线 | 每5格加粗边框，方便定位行列位置 |
| 💡 柔和高亮 | brightness + 暖色 drop-shadow 替代粗白框，无割裂感 |
| 🏷️ 色号进度 | 从逐格点击改为按色号勾选，符合"一种颜色一种颜色放"的实际制作流程 |

## 🖼️ 界面说明

### 工具栏
生成图纸后，顶部工具栏提供以下操作：
- **翻转** — 水平镜像翻转
- **去杂色** — 弹出面板选择轻度/中度/重度
- **品牌转换** — 下拉切换色系套装（72/96/120/144/221）
- **导出** — PNG / PDF / JSON 三种格式

### 图纸交互
- **点击格子** — 自动高亮该色号，同色格子发光，其余变暗
- **滚轮/捏合** — 缩放图纸
- **拖拽** — 平移查看
- **小地图** — 右下角缩略图，点击跳转

### 侧边栏
- **色号统计** Tab — 每个色号的用量、百分比、勾选框（标记完成）
- **材料清单** Tab — 按色系分组，含勾选框，方便采购对照

## 📊 埋点统计

系统自动记录行为数据（图片生成、AI生成、导出、板型/色系偏好、活跃时段），存储在本地 `perler_analytics/` 目录（JSONL格式）。

统计接口：
```
GET /api/analytics/overview      # 总览
GET /api/analytics/daily         # 每日趋势
GET /api/analytics/board-usage   # 板型分布
GET /api/analytics/color-usage   # 色系分布
```

## ⚙️ 配置

通过 `config.py` 或环境变量配置：

```python
WX_APPID = os.getenv('WX_APPID', '')        # 微信开放平台AppID
WX_SECRET = os.getenv('WX_SECRET', '')       # 微信Secret
JWT_SECRET = os.getenv('JWT_SECRET', '...')  # JWT签名密钥
JWT_EXPIRY_HOURS = 168                       # Token有效期（7天）
GUEST_DAILY_LIMIT = 3                        # 游客每日生成次数
USER_DAILY_LIMIT = 50                        # 注册用户每日生成次数
```

## ⚠️ 注意事项

1. 上传图片越清晰，生成效果越好；过大会自动压缩
2. AI文字生成需要网络连接，离线环境自动降级为像素画模式
3. PDF导出支持中文，自动适配系统字体
4. 游客数据每日凌晨重置
5. 微信登录需配置微信开放平台应用
6. 进度标记数据保存在浏览器本地存储（localStorage），换设备/清缓存会丢失
